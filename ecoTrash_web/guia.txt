Preparar el entorno virtual (Windows + PowerShell) y dependencias

En PowerShell (estando en la ruta del proyecto):

Crear entorno virtual:

python -m venv env


Si PowerShell bloquea activar el script por políticas:

Ejecuta PowerShell como Administrador una sola vez y permite scripts para tu usuario:

Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned


(Confirma Y si te lo pide.)

Activar entorno en PowerShell:

.\env\Scripts\Activate.ps1


Si usas cmd.exe: .\env\Scripts\activate.bat

Instalar dependencias mínimas:

pip install --upgrade pip
pip install flask pillow numpy tensorflow

tensorflow puede tardar y ocupar mucho. Si ya tienes un ambiente que funciona, usa ese. Si tu CPU es Intel/AMD, la versión pip install tensorflow instalará TF con aceleración oneDNN.

Si pip install tensorflow da errores por espacio o versión, dinos el error y ajustamos.

python app.py --host 0.0.0.0 --port 5000
